MEMORY GAME

Game Rules

● The game will start as soon as you click the start button.
● When clicking the 'Start Game' button, the cards will be randomly shuffled.
● After 2 seconds, the cards will be turned face down.
● You must click on the cards in ZSENA order.
● Each card you click in the correct order earns you 20 points, you can earn a total of 100 points.
● If you click the wrong row of cards the game will end (e.g. clicking 'N' after 'Z').
● You can also click a 'Restart' button to play the game again.
